# Blognet

> A blog written in ASP.NET core.

## Install
First clone the project
```
git clone https://github.com/atila29/blognet.git
```
cd into the directory
```
cd blognet
```
Restore
```
dotnet restore
```
To build run
```
dotnet build
```
## Usage

```
dotnet run /src/dtu.blognet.Application.Web/dtu.blognet.Application.Web.csproj
```
